Juno

Implementing a virtual machine in one week

Juno is a partial, stack-based, virtual machine that supports generic functions, first-order functions, control-flow, classes and super-classes, int64 numbers, strings, a few builtins, and a powerful runtime. I wrote it during a rainy, boring week. Finally I wrote thins article to show you how you can create your own language during one week. The vm is created in less than 660 lines of hand-written JavaScript.
